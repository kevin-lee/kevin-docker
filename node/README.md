# Node

* Build locally
  ```shell
  docker build -t node:main .
  ```

* Run locally in interactive mode
  ```shell
  docker run -i -t node bash
  ```

* Pull the image
  ```shell
  docker pull ghcr.io/kevin-lee/node:main
  ```

* Run in interactive mode
  ```shell
  docker run -i -t ghcr.io/kevin-lee/node:main bash
  ```
